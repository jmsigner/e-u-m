Prüfungsleistungen

Veröffentlichungsdatum

12. April 2026

Die Benotung für dieses Modul erfolgt anhand einer strukturierten Hausarbeit. Um für die Hausarbeit zugelassen zu werden, muss eine Prüfungsvorleistung absolviert werden, darin werden die wichtigsten Konzepte, Methoden und Begriffe der Vorlesung in einem Multiple-Choice-Test abgefragt.

Für die Hausarbeit verwenden Sie einen bereits erhobenen Datensatz. Es ist auch zulässig eine beretis durchgeführte Studie zu wiederholen. Nähere Details zur Hausarbeit fogen ab Woche 8 des Semesters.

Statistik

Es werden keine R-Kenntnisse abgefragt, lediglich die Interpretation von Ergebnissen.

Prüfungsleistung

Die Prüfungsleistung besteht aus einer strukturierten Hausarbeit.

Stellen Sie eine Hypothese/Erwartung auf und analysieren Sie dazu einen bereits vorhandenen Datensatz.

  • Analysen (40%)
  • Schriftliche Ausarbeitung (60%)

Analysen

  1. Code ist reproduzierbar. (60%)
  2. Der Code ist ausreichend dokumentiert und logisch sinnvoll aufgebaut. (25%)
  3. Es gibt eine README-Datei, die beschreibt, wie man den Code und die Analyse ausführen kann. (15%)

Hausarbeit

Was Bewertung Detaillierte Bewertung
Einleitung 15 % 1. Worum geht es (1–2 Absätze); 30 %
2. Stand der Dinge (1 Absatz); 20 %
3. Ziel dieser Hausarbeit; 20 %
4. Literatur (3–5 peer-reviewed Veröffentlichungen); 30 %
Methoden 25 % 1. Datenquellen, wie wurden die Daten aufgenommen; 20 %
2. Modellbeschreibung (welches Modell wurde verwendet, wie wurde es parametrisiert); 40 %
3. Aufschreiben und beschreiben der Modellgleichung; 30 %
4. Richtige Zitation aller Software; 10 %
Ergebnisse 25 % - Beschreibung der Ergebnisse, die für die Fragestellung relevant sind; 20 %
- Zwei Abbildungen, die die Ergebnisse untermauern und das Modell visualisieren; 60 %
- Eine Tabelle, die für die Fragestellung relevante Rohdaten oder Modelloutput zusammenfasst 20 %
Diskussion 15 % Jeweils ein Absatz, der:
1. der, die Ergebnisse zusammenfasst und in Verbindung mit den Hypothese(n) bringt; 30 %
2. der, mögliche Schwachstellen, Erweiterungen etc. diskutiert; 30 %
3. der, die Ergebnisse im Kontext der aktuellen begutachteten Literatur (>2020) darstellt (3–5 Referenzen); 40 %
Referenzen 5 % Alle verwendeten Quellen (inkl. Software) sind in einem Literaturverzeichnis einheitlich zitiert
Zusammenfassung 5 % Dem Text ist eine Zusammenfassung von einem Absatz vorangestellt (max. 200 Wörter)
Form 5 % Die Arbeit ist einheitlich und ansprechend formatiert
Ausdruck 5 % Die Arbeit ist verständlich und fehlerfrei geschrieben
